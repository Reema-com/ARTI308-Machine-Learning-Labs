# Lab 3 – Exploratory Data Analysis (EDA)

# Objective: 
The aim of this lab is to perform Exploratory Data Analysis (EDA) on the Students Performance dataset. The purpose is to understand the structure of the data, identify patterns, and extract meaningful insights.

# Dataset
Dataset: Students Performance in Exams  
Source: Kaggle  

The dataset contains 1000 student records and 8 features including demographic information and subject scores (math, reading, and writing).

# Tasks Performed
In this lab, the following EDA techniques were applied:
- Inspecting the data structure (shape, columns, data types)
- Checking for missing values
- Checking for duplicate records
- Generating summary statistics
- Performing univariate analysis (distribution of individual variables)
- Performing bivariate analysis (relationships between variables)
- Creating visualizations such as histograms, boxplots, countplots, scatterplots, and heatmaps

# Key Insights
- Students' scores follow an approximately normal distribution.
- Reading and writing scores are strongly correlated.
- Test preparation course completion is associated with higher performance.
- Socioeconomic factors (lunch type) may influence academic performance.

# Conclusion
This lab demonstrates how EDA helps in understanding datasets before applying further analysis or machine learning models. Visualizations and statistical summaries provide valuable insights into patterns and relationships within the data.
